# Privacy Policy

_Last update: 2026-01-21_

This application respects your privacy and is designed to collect only the data strictly necessary for its operation.

---

## 1. Data Collection

When using the application, the following data may be collected:

- **Account information**  
  Authentication is performed using **Sign in with Apple**.  
  We may receive an email address and a username associated with your Apple account.

- **User-generated content**  
  Photos of recipes provided by the user for processing.

---

## 2. Use of Data

Collected data is used exclusively to:

- Provide the core features of the application
- Process recipe photos using an artificial intelligence service
- Allow optional synchronization of user data across devices

Photos sent for AI processing are **used only for analysis and are deleted immediately after processing**.  
No images are stored or reused for any other purpose.

---

## 3. Third-Party Services

The application may rely on the following services:

- **Apple services**  
  Authentication and App Store Analytics

- **Artificial intelligence service**  
  Photos are processed using **Claude by Anthropic**.  
  Images are transmitted without any association to the userâ€™s identity and are deleted after processing.

- **Firebase**  
  Used only if cloud synchronization is enabled by the user.

Each third-party service processes data in accordance with its own privacy policy.

---

## 4. Data Storage and Security

User data is stored locally on the device by default.

If cloud synchronization is enabled, data is stored on secure servers.

Reasonable technical and organizational measures are implemented to protect user data.

---

## 5. User Rights

You may:

- Access your personal data
- Request deletion of your account and associated data
- Disable optional features such as cloud synchronization
