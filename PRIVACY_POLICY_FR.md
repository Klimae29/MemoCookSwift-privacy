# Politique de confidentialité

_Dernière mise à jour : 21 janvier 2026_

Cette application respecte votre vie privée et est conçue pour collecter uniquement les données strictement nécessaires à son fonctionnement.

---

## 1. Données collectées

Lors de l’utilisation de l’application, les données suivantes peuvent être collectées :

- **Informations de compte**  
  L’authentification est réalisée via **Se connecter avec Apple**.  
  Nous pouvons recevoir une adresse e-mail et un pseudonyme associés à votre compte Apple.

- **Contenu généré par l’utilisateur**  
  Photos de recettes fournies par l’utilisateur pour traitement.

---

## 2. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

- Fournir les fonctionnalités principales de l’application
- Traiter les photos de recettes à l’aide d’un service d’intelligence artificielle
- Permettre la synchronisation optionnelle des données entre appareils

Les photos envoyées pour le traitement par l’IA sont **utilisées uniquement pour l’analyse et supprimées immédiatement après le traitement**.  
Aucune image n’est stockée ou réutilisée à d’autres fins.

---

## 3. Services tiers

L’application peut s’appuyer sur les services suivants :

- **Services Apple**  
  Authentification et App Store Analytics

- **Service d’intelligence artificielle**  
  Les photos sont traitées à l’aide de **Claude (Anthropic)**.  
  Les images sont transmises sans être associées à l’identité de l’utilisateur et sont supprimées après traitement.

- **Firebase**  
  Utilisé uniquement si la synchronisation cloud est activée par l’utilisateur.

Chaque service tiers traite les données conformément à sa propre politique de confidentialité.

---

## 4. Stockage et sécurité des données

Les données utilisateur sont stockées localement sur l’appareil par défaut.

Si la synchronisation cloud est activée, les données sont stockées sur des serveurs sécurisés.

Des mesures techniques et organisationnelles raisonnables sont mises en œuvre afin de protéger les données utilisateur.

---

## 5. Droits de l’utilisateur

Vous pouvez :

- Accéder à vos données personnelles
- Demander la suppression de votre compte et des données associées
- Désactiver les fonctionnalités optionnelles telles que la synchronisation cloud
